FAQ

What visitors usually ask first.

Will this show up on Google?

Yes. Because the site is publicly hosted and includes structured page content, search engines can index it over time.

Is this free to host?

Yes. The website is hosted for free with GitHub Pages, while the live detector continues to run separately on Streamlit.

Will it still sleep?

The website itself stays available continuously. The Streamlit demo may still require a short wake-up time after inactivity on the free tier.

Why keep both the website and the app?

The website provides a polished public-facing presence, while the detector app handles the heavier backend processing and model inference.
