FAQ

What visitors usually ask first.

Will this show up on Google?

Yes. A public GitHub Pages site can be indexed by Google, although indexing may take days or weeks.

Is this free to host?

Yes. GitHub Pages can host this landing page for free. Your detector can stay on Streamlit for now.

Will it still sleep?

The GitHub Pages website will not sleep. The separate Streamlit detector can still sleep on the free tier after inactivity.

Why keep both the website and the app?

The website gives you a professional public presence, while the detector app handles the heavy Python model work.
